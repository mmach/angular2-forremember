This application will help you to manage the containers, locations and samples in the laboratory.

Please login or register if you don't have account yet!

Locations

Find the information about all locations available in the system. Locations tree contains information about country, city, building and particular freezer. You can create new or edit existing location or freezer.

Containers

View the list of all containers available in the system. Select container using searching functionality. A new container, or a batch of containers by providing bar codes range.

Concepts

View the list of all concepts created in system. You can create a new concept and register existing one.
